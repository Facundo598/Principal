📡 ESP8266 + Firebase + GitHub Pages

Sistema para controlar o visualizar datos en tiempo real usando ESP8266, Firebase y una página web.

🔄 Flujo del sistema

ESP8266 → Firebase Realtime Database → Página Web
El ESP envía o lee datos desde Firebase, y la web los consume en tiempo real.

🔥 PARTE 1 — Crear proyecto en Firebase

  1. Ir a Firebase Console
  2. Click en Crear proyecto
  3. Asignar nombre
  4. Desactivar Analytics (opcional)

🔥 PARTE 2 — Crear Realtime Database

  1. Ir a Build → Realtime Database
  2. Click en Crear base de datos
  3. Elegir región
  4. Seleccionar modo prueba o bloqueado
